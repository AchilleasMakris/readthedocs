# Δομές Ελέγχου Ροής

## **ΕΑΝ**

Η δομή **ΕΑΝ**  χρησιμοποιείται για να ελέγξει αν μια συνθήκη είναι αληθής και, αν είναι, να εκτελέσει το μπλοκ εντολών που ακολουθεί.
Η δομή **ΑΛΛΙΩΣ** μπορεί να χρησιμοποιηθεί για να εκτελέσει έναν άλλο μπλοκ εντολών αν η συνθήκη είναι ψευδής.
```
ΕΑΝ <συνθήκη> ΤΟΤΕ
    <εντολές αν αληθής>
ΑΛΛΙΩΣ
    <εντολές αν ψευδής>
ΕΑΝ-ΤΕΛΟΣ
```

### Παράδειγμα χρήσης του **ΕΑΝ**
```
ΕΑΝ (X < 5) ΤΟΤΕ
    Y:= Y * X;
ΑΛΛΙΩΣ
    X:= X + 1;
ΕΑΝ-ΤΕΛΟΣ
```
### Παράδειγμα χρήσης του με λογικό **AND**
```
ΕΑΝ (X > 0) AND (X < 5) ΤΟΤΕ
    Y:= Y * X;
    X:= X + 1;
ΑΛΛΙΩΣ
    Y:= Y + X;
ΕΑΝ-ΤΕΛΟΣ
```
